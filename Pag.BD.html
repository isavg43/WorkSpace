Base de Datos y CRUD

¿Qué es una base de datos?

Una base de datos es un sistema organizado para guardar, gestionar y acceder a información usando una computadora. Permite almacenar muchos datos y encontrarlos fácilmente cuando se necesitan.

¿Para qué sirve una base de datos?

Sirve para guardar y organizar la información que usa un programa o una aplicación.

Por ejemplo, aplicaciones como WhatsApp o Instagram guardan usuarios, mensajes y fotos en bases de datos.

¿Qué es un CRUD?

Un CRUD es un conjunto de acciones básicas para manejar datos en una base de datos.

  • C – Create: crear datos.
  • R – Read: leer datos.
  • U – Update: actualizar datos.
  • D – Delete: eliminar datos.

¿Qué es un campo?

Un campo es el espacio donde se guarda un dato dentro de una tabla.

¿Qué es un registro?

Un registro es una fila de datos dentro de una tabla.

¿Qué es un dato?

Un dato es la información que se guarda en una base de datos.

¿Qué es un diagrama entidad-relación?

Es un dibujo que muestra cómo se organizan y se relacionan los datos en una base de datos.

Entidad-Relación

Un diagrama entidad-relación (DER) es un esquema que se usa para representar cómo se organizan los datos en una base de datos y cómo se relacionan entre sí. En el diagrama se muestran entidades (como proveedor, empleado, proyecto o pieza), sus atributos (nombre, número, edad, etc.) y las relaciones entre ellas (por ejemplo trabajo, suministro o inventario).
